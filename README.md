# Welcome to WP Like Plugin!

WP Like Plugin created to help admins to analyze which tag attracts users more.

# Features
•Users can like posts if they like and admins be aware of which post liked.
•Admin can add widget to your wordpress to show users which post liked most.
•Multiple likes forbidden and protected with ip address check system.

# Installation

To install wordpress plugin is the same with standard wordpress plugin installation.
You just need to find your wordpress directory copy like-plugin folder inside to
**\wp-content\plugins** folder.
Then open wordpress admin panel click Plugins you should see **Like Counter** plugin there.
Click activate button. You successfully installed the plugin.
